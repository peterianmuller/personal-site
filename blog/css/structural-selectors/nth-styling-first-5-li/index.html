A few different ways to style the first 5 li elements in a list

These ideas are based on Estelle Weyl's excellent lectures on selectors: .

Let's say we have a list and want to style the first 5 li elements. One way we could do this if we don't know how many list items we have nested inside of our li is by first adding a class to all li elements with our desired style. We could then select all li elements after a certain nuber using :nth-of-type(n + 6), and removing that class.

ul li {
font-weight: bold;
}
ul li:nth-of-type(n + 6) {
font-weight: normal;
}

That'll do the trick! However, if we knew how many list items were in the list and it was unchanging, we cold use :nth-last-of-type to start at the end of the list and move up to the 5th list item. We would then increment n, but this would be pushing the styling 1 "up" or away from the last-of-type. We would eventually end at the farthest away from the last of type, which is of course the first-of-type, or the beginning of our list

ul li:nth-last-of-type(n + 6) {
font-weight:bold;
}
