I've seen many sites that have navigaton links on the left-hand side of the top of the screen. Thus I went ahead and made a simple left-hand nav bar.

In order to do this, we can use a ul tag and remove the dots that are given by default to each list item by activating the list-style: none rule. We then float each li to the left and remove some of the default link styling by selecting each link tag inside of the li.

* {

margin: 0;

padding: 0;

}


ul#nav {

list-style: none;

}


ul#nav li {

float: left;

}


ul#nav li a {

text-decoration: none;

font-weight: bold;

padding: 0 5px;

}

Now, setting a float takes an element outside of the normal flow od the DOM, thus its important to clear the float after we are finished floating elements and would like to move back to the regular flow of the page. We can go ahead and place a div after the ul with a class of clear : both
